DittoAndroidTools 6 supports Ditto Kotlin SDK 5 and requires `com.ditto:ditto-kotlin` 5.0.1 or a later 5.x release.

## Breaking changes

- Install the tools from `com.ditto:ditto-tools-android` and update tools imports from `live.ditto.tools.*` to `com.ditto.tools.*`.
- Ditto Kotlin SDK 4 is no longer supported. The SDK dependency is now `com.ditto:ditto-kotlin`, and SDK imports use `com.ditto.kotlin.*`.
- The minimum Android API level increased from 23 to 24. The standalone project now compiles against Android API 36.
- Ditto initialization, observers, subscriptions, store values, and exception types use the SDK 5 APIs.
- Presence Degradation conversion extensions now receive `DittoCborSerializable.Dictionary` instead of the SDK 4 `DittoDocument` type.
- Disk Usage uses `DittoDiskUsageItem` instead of `DiskUsageItem`; `DiskUsage.sizeInBytes` and `DiskUsageState.totalSizeInBytes` are now `Long` values.
- Heartbeat `addToCollection` is now a suspending function because its SDK 5 DQL write is asynchronous.
- Data Browser collections are exposed as collection-name strings instead of SDK collection objects. The obsolete `isStandAlone` arguments and state were removed from `ListItem`, `DocumentsViewModel`, and its factory.

## Heartbeat changes

- Heartbeat documents add a `connectionTypes` map containing every observed SDK connection type. The existing `bluetooth`, `p2pWifi`, and `lan` summary fields remain available.

## Added

- Added the Log Viewer with file selection, tailing, filtering, log-level display, details, and scroll controls.

## Fixed

- Fixed Data Browser behavior with large datasets, collection discovery, DQL collection-name quoting, CBOR map and list display, and standalone subscriptions.
- Fixed leaked observers, log-path handling, disk-usage path display, and Heartbeat serialization during the v5 migration.
- Fixed Heartbeat and the Presence Degradation Reporter incorrectly counting WebSocket connections as LAN.
- Restored the Presence Viewer assets used by previous GA tools releases. (SDKS-4488)
